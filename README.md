# Personal Website

A personal website and portfolio built to showcase who I am both in and out of work. This site is designed to introduce me as a developer, highlight my projects, present my services, and make it easy for potential clients, employers, and collaborators to connect with me.

## Overview

This website combines a personal brand site, developer portfolio, and freelance services site into one experience. It is meant to help visitors quickly understand:

- who I am
- what I build
- what services I offer
- how to contact me

The site includes sections for my background, hobbies, tech stack, projects, services, testimonials, and contact information.

## Goals

- Create a professional online presence
- Share my software development journey
- Showcase projects with live demos and GitHub repositories
- Offer web development services with clear pricing and packages
- Build trust through testimonials
- Provide an easy way for visitors to contact me

## Pages

### Home

The landing page introduces me with:

- my name
- my title
- a short value statement
- a background video or rotating photos
- call-to-action buttons

### About

The about page includes:

- a paragraph about my software development journey
- a photo of me
- hobbies and interests
- programming languages and tech stack

### Projects

The projects page features:

- a grid of project cards
- project image
- project name
- short description
- GitHub repository link
- deployed/live site link

### Services

The services page outlines:

- single-page application pricing
- 3-page static website pricing
- custom website pricing
- what is included
- process/workflow
- a call-to-action to get in touch

### Testimonials

The testimonials page displays:

- verified client reviews
- project-related feedback
- reviews submitted through private links to reduce fake submissions

### Contact

The contact page provides:

- a contact form
- professional email
- social/profile links
- availability for freelance or job opportunities
